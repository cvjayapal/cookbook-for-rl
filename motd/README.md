# jnj_motd Cookbook
  
  Installs message of the day

## Requirements

### Platforms

- CentOS_7.1/7.2
- Ubuntu_14.04

### Chef

- Chef 12.0 or later


## Attributes
 
 

## Usage

### jnj_motd::default

Just include `jnj_motd` in your node's `run_list`:

{
  "name":"my_node",
  "run_list": [
    "recipe[jnj_motd]"
  ]
}
